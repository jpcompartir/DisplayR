# Package index

## Plotting in R - ggplot2

Functions built upon the {ggplot2} R Package

- [`dr_plot_vot()`](dr_plot_vot.md) : Quickly create a bar chart which
  shows volume over time.
- [`dr_plot_vot_group()`](dr_plot_vot_group.md) : Quickly make a line
  plot of volume over time for a grouping variable
- [`dr_plot_sent()`](dr_plot_sent.md) : Create a sentiment distribution
  bar chart
- [`dr_plot_sent_group()`](dr_plot_sent_group.md) : Plot sentiment
  grouped by a certain variable.
- [`dr_plot_sent_vot()`](dr_plot_sent_vot.md) : Plot a bar or line chart
  of sentiment over time
- [`dr_plot_raincloud()`](dr_plot_raincloud.md) : Quickly create a
  raincloud plot

## Tabular Data

gt and flextable functions for creating beautiful tables in R

- [`disp_flextable()`](disp_flextable.md) : Create a flextable for
  volume + sentiment with a gradient fill.
- [`disp_gt()`](disp_gt.md) : Create a gt_summary table fit for use in
  client briefs

## PowerPoint Plotting

Functions for exporting common & basic charts to PowerPoint

- [`disp_ms_vot()`](disp_ms_vot.md) : Display Time Series Chart of Data
  Volume
- [`disp_ms_vot_grouped()`](disp_ms_vot_grouped.md) : Display Time
  Series Chart of Grouped Data Volume
- [`disp_ms_sent_grouped()`](disp_ms_sent_grouped.md) : Display a
  Stacked Bar Chart of Sentiment Grouped by a Variable
- [`dr_mmlb_barchart()`](dr_mmlb_barchart.md) : Create a stacked bar
  chart for Microsoft's Making My Life Better classification

## PowerPoint Utility Functions

Utility functions for working with PowerPoint from R

- [`disp_add_slide()`](disp_add_slide.md) : Add a new slide to a
  PowerPoint presentation and insert a chart
